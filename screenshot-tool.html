How to Capture Screenshots:

  1. Select a device size below
  2. Wait for the app to load in the frame
  3. Navigate to the screen you want to capture
  4. Press F12 to open Developer Tools
  5. Click the frame, then press Ctrl+Shift+P
  6. Type screenshot and select "Capture node screenshot"
  7. Or use Windows Snipping Tool (Win+Shift+S) to capture the frame
